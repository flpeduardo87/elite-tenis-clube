# ATUALIZAR NA VERCEL

3. **Settings** > **Environment Variables**

4. EDITE ou ADICIONE estas variáveis:

   **VITE_SUPABASE_URL**
   - Production: [Cole a URL do Supabase ATUAL]
   - Preview: [Cole a URL do Supabase ATUAL]
   - Development: [Cole a URL do Supabase ATUAL]

   **VITE_SUPABASE_ANON_KEY**
   - Production: [Cole a anon key do Supabase ATUAL]
   - Preview: [Cole a anon key do Supabase ATUAL]
